OpenData SMN

A simple API for SwissMetNet data

OpenData SMN is a simple REST API for SwissMetNet data. It's developed in the open supporting the OpenData.ch initiative.

The actual data for 10min intervals is provided by opendata.swiss, the Swiss open government data portal.

Getting started

  • navigate to the /smn resource to fetch the latest available data set
  • browse the executable API documentation to discover the full API
  • download a generated OpenAPI v3 YAML contract
